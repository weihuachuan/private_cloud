# private_cloud
私有云、网盘、hadoop伪分布式、HDFS+HBASE+MYSQL存储、Node.js。<br>项目使用前后端分离技术分。

## 系统环境和依赖
* centos7
* mysql_5.7
* jdk1.8
* hadoop伪分布式
* zookeeper
* hbase
* node.js
